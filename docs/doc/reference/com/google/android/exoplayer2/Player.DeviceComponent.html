      • getDeviceVolume

        int getDeviceVolume()
        Gets the current volume of the device.

        For devices with local playback, the volume returned by this method varies according to the current stream type. The stream type is determined by AudioAttributes.usage which can be converted to stream type with Util.getStreamTypeForAudioUsage(int). The audio attributes can be set to the player by calling Player.AudioComponent.setAudioAttributes(com.google.android.exoplayer2.audio.AudioAttributes, boolean).

        For devices with remote playback, the volume of the remote device is returned.
