      • create

        @Capabilities
static int create​(@FormatSupport
                  int formatSupport)
        Returns RendererCapabilities.Capabilities for the given C.FormatSupport.

        The RendererCapabilities.AdaptiveSupport is set to ADAPTIVE_NOT_SUPPORTED and {RendererCapabilities.TunnelingSupport is set to TUNNELING_NOT_SUPPORTED.

        Parameters:
        formatSupport - The C.FormatSupport.
        Returns:
        The combined RendererCapabilities.Capabilities of the given C.FormatSupport, ADAPTIVE_NOT_SUPPORTED and TUNNELING_NOT_SUPPORTED.
