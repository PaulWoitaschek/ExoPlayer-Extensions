    Forces the selection of trackIndices for a TrackGroup.

    If multiple tracks in trackGroup are overridden, as many as possible will be selected depending on the player capabilities.

    If trackIndices is empty, no tracks from trackGroup will be played. This is similar to TrackSelectionParameters.disabledTrackTypes, except it will only affect the playback of the associated TrackGroup. For example, if the only C.TRACK_TYPE_VIDEO TrackGroup is associated with no tracks, no video will play until the next video starts.
