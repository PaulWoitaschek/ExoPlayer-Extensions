Class TrackSelectionOverrides

    Forces the selection of the specified tracks in TrackGroups.

    Each override only affects the selection of tracks of that type. For example overriding the selection of an audio TrackGroup will not affect the selection of video or text tracks.

    If multiple TrackGroups of the same C.TrackType are overridden, which tracks will be selected depend on the player capabilities. For example, by default ExoPlayer doesn't support selecting more than one TrackGroup per C.TrackType.

    Overrides of TrackGroup that are not currently available are ignored. For example, when the player transitions to the next MediaItem in a playlist, any overrides of the previous MediaItem are ignored.
